# @adonis0123/code-doc-generator

Claude Code 技能包 - 自动分析代码仓库并生成包含 Mermaid 流程图的 README.md 文档。

## 功能

- 自动分析项目结构和代码架构
- 生成包含 Mermaid 图表的 README.md
- 支持多种编程语言（JavaScript/TypeScript、Python、Java、Go）
- 智能更新现有文档

## 安装

```bash
npm install -g @adonis0123/code-doc-generator
```

## 使用

安装后，在 Claude Code 中使用以下命令：

```
/code-doc-generator
```

或直接描述需求：

```
"为这个项目生成包含架构图的文档"
"更新 README 中的 API 文档部分"
"添加一个展示组件交互的架构图"
```

## 包含内容

- `SKILL.md` - 技能定义
- `references/mermaid-syntax.md` - Mermaid 语法参考
- `references/language-patterns.md` - 语言分析模式
- `references/readme-templates.md` - README 模板示例
